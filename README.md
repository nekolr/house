## 简介
一个简单的爬虫程序，目的是寻找北京的住房。

## 功能简介
从所有可能的房源信息网抓取数据，进行简单的数据清洗后本地保存，对外提供查询、图表展示、价格趋势、价格对比等功能。

## 数据来源
- 链家
- 我爱我家
- 安居客
- 贝壳
- 58 同城
- 赶集
- 吉屋
